# P02 — Churn Prediction API

Production-style churn scoring API with audit log and banker dashboard.

**Resume anchor:** Predictive modeling for customer churn and engagement analysis (Willamette Valley Bank).

## Stack

| Layer | Tech | Version |
|-------|------|---------|
| API | FastAPI, scikit-learn, SQLite | FastAPI 0.115, sklearn 1.6 |
| Frontend | React, Vite | React 19 |

> Uses **SQLite** for the audit log (mentor step allows SQLite or Postgres). Swap to Postgres later for portfolio depth.

## Quick start (9 commands)

```powershell
# 1 — backend setup
cd apps/p02-churn-api/backend
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt

# 2 — train model (writes artifacts/ + data/churn_training.csv)
python train.py

# 3 — run API on port 8001 (P01 uses 8000)
uvicorn main:app --reload --port 8001

# 4 — new terminal, frontend on 5174 (P01 uses 5173)
cd apps/p02-churn-api/frontend
npm install
npm run dev
```

- Dashboard: **http://localhost:5174**
- OpenAPI docs: **http://localhost:8001/docs**

## API endpoints

| Method | Path | Description |
|--------|------|-------------|
| GET | `/health` | API + DB + model status |
| GET | `/model/metadata` | AUC, version, feature list |
| POST | `/predict` | Single customer churn score |
| POST | `/predict/batch` | Up to 500 customers |
| GET | `/predictions` | Paginated audit history |

### Example predict payload

```json
{
  "customer_id": "CUST-9001",
  "tenure_months": 8,
  "monthly_charges": 85.5,
  "total_charges": 620.0,
  "contract_type": "month",
  "support_calls": 4
}
```
