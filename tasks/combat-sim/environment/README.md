# Combat Simulator Data

- `data/characters/` — character definitions (stats, element, abilities)
- `data/battles/` — battle scenarios (teams and per-round action plans)
- `data/config.json` — global configuration
- `data/element_chart.json` — elemental type matchup chart
- `data/rules.md` — complete combat rules
- `data/output_format.md` — required output JSON structure
- `data/damage_formula.md` — detailed damage calculation steps
- `data/turn_order.md` — turn ordering rules
