Title: Reverse Words in a String (TypeScript)

Description:
Implement `reverseWords(s: string): string` which returns a new string with the order of the words reversed.

Requirements:
- Words are sequences of non-space characters.
- Trim leading/trailing spaces and collapse multiple spaces to a single space.
- Preserve word characters and order within each word.

Examples:
- Input: "  hello   world  "  -> Output: "world hello"
- Input: "a good   example"  -> Output: "example good a"

Edge cases:
- Empty or all-space strings -> return empty string.

Place your TypeScript implementation in `solution/index.ts`. Tests are in `tests/` and the project builds with `npm test` at the task root.
