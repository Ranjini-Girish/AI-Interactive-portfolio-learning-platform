Emit five UTF-8 JSON files under `/app/audit/` named `module_catalog.json`, `linear_order.json`, `cycle_members.json`, `path_weights.json`, and `summary.json`. The full ordering contract, SCC membership rule, path-weight definition, and canonical JSON presentation are spelled out in `/app/ivybuild/SPEC.md`. Evidence lives under `/app/ivybuild/`; a non-empty `IBL_DATA_DIR` overrides reads and a non-empty `IBL_AUDIT_DIR` overrides writes, otherwise those defaults apply. Create the audit directory when missing and never rewrite the evidence tree.
