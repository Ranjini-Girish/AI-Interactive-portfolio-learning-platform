The Java 21 project at `/app` needs a spectral analysis tool that reads sensor time-series data from `/app/data/sensor_timeseries.csv`, estimates the power spectral density using Welch's method, detects spectral peaks, and writes a JSON report to `/app/output/spectral_report.json`.

The analysis parameters are in `/app/config/analysis_params.json` and `/app/config/peak_detection.json`. Window function details and PSD normalization are documented in `/app/docs/spectral_analysis.md`, and the full Welch's method algorithm including segmentation, one-sided spectrum conversion, peak interpolation, and the output JSON schema are in `/app/docs/welch_method.md`. An abbreviated example of the expected output structure is at `/app/data/example_output.json`. All floating-point values in the output must be rounded to 6 decimal places, with sorted keys at every nesting level, 2-space indentation, and a trailing newline.
