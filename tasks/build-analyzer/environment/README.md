# Build Analyzer Data

- `data/targets/` — build target definitions (one JSON per target)
- `data/config.json` — project configuration
- `data/changes.json` — list of changed source/header files
- `data/build_spec.md` — build system specification
- `data/rebuild_rules.md` — dirty target detection and rebuild ordering
- `data/parallel_rules.md` — parallel level assignment and timing
- `data/cycle_detection.md` — cycle detection algorithm and reporting
- `data/output_format.md` — required output JSON structure
