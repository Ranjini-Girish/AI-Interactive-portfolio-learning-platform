Site reliability filed a regression ticket asking for a deterministic latch replay over the frozen signal bundle under `/app/htl_lab/`. The replay must stay read-only on that tree, honor the hysteresis thresholds and margin spelled in `/app/htl_lab/SPEC.md`, and emit the UTF-8 JSON filenames listed there with the canonical pretty-printer from the same document. Create `/app/audit/` when it does not exist yet.

Incident objects may bias samples on bounded day ranges or temporarily force the latch low even when the unbiased signal would cross the high threshold; the specification defines the order those adjustments apply relative to the hysteresis update. Non-empty `HTL_DATA_DIR` selects a different read root than `/app/htl_lab/`, and non-empty `HTL_AUDIT_DIR` selects a different write root than `/app/audit/`; unset or empty variables keep the defaults. Anchors and ancillary JSON remain part of the integrity bundle.
