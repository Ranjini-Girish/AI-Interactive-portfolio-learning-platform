Operations control wants a frozen read-only audit of the slot backlog bundle under `/app/sbm_lab/` without touching any bytes in that tree. The audit must emit only the UTF-8 JSON filenames enumerated in `/app/sbm_lab/SPEC.md`, using the canonical pretty-printer defined there, and it must create `/app/audit/` when the directory is absent. The specification is the single source for merge slack, lane ordering, bitmask eligibility, incident windows tied to the ledger day, and the exact sorting keys for grouped spans.

If `SBM_DATA_DIR` is non-empty it becomes the read root instead of `/app/sbm_lab/`; if `SBM_AUDIT_DIR` is non-empty it becomes the write root instead of `/app/audit/`; unset or empty variables keep those defaults. Anchors under `/app/sbm_lab/anchors/` and ancillary JSON under `/app/sbm_lab/ancillary/` are operator notes but still participate in integrity checks alongside the numbered item records.
