Implement a statistical analysis pipeline in TypeScript (TypeScript on Node 22) that reads a pipeline configuration from `/app/data/pipeline_config.json` and eight experiment datasets from `/app/data/datasets/`, applies descriptive statistics, outlier detection, pairwise hypothesis testing, effect size estimation, bootstrap confidence intervals, and rank correlation to each experiment, then writes a comprehensive audit report to `/app/output/stats_report.json`. The environment provides TypeScript on Node 22 with no additional npm packages installed; the project manifest is at `/app/package.json`. Do not modify any files under `/app/data/`.

Each experiment file contains named groups of numerical observations. The pipeline computes per-group descriptive statistics including fractional trimmed means, MAD-based outlier detection with modified Z-scores, Welch's t-tests with exact degrees of freedom for all group pairs, Hedges' g effect sizes with the small-sample correction factor, bootstrap percentile confidence intervals using a deterministic Park-Miller LCG seeded from the config, and Spearman rank correlations with proper tie handling. The complete specification for each computation, including formulas, edge-case behavior, and the output structure, is documented in the `.md` files under `/app/docs/`.

The output JSON uses 2-space indentation with a trailing newline. It contains a config copy, an experiments array sorted by experiment_id with per-group statistics and pairwise comparisons sorted lexicographically, and a summary with aggregate counts. All floating-point values are rounded to the number of decimal places specified in the configuration.
