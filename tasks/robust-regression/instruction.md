The robust regression pipeline at `/app/src/` implements Iterative Reweighted Least Squares (IRLS) with the Huber M-estimator to fit a linear model to sensor data, compute robust standard errors via the sandwich covariance estimator, and detect statistical outliers. The implementation contains multiple correctness bugs across different modules — in the numerical algorithms, data handling, and output formatting — that cause wrong regression coefficients, incorrect standard errors, missed outliers, wrong convergence behavior, and inaccurate diagnostics.

Fix all bugs so that running `cargo build --release && /app/target/release/robust-regression` from `/app` produces a correct JSON report at `/app/output/analysis.json`. The pipeline's behavioral contract, including the exact mathematical formulas for each algorithm component, is specified in `/app/docs/SPEC.md`. All floating-point values in the output must be rounded to the precision specified in `/app/config/analysis.toml`.
