The TypeScript on Node 22 project at `/app` needs a dependency graph audit tool. It reads a package registry from `/app/data/registry.json` and security advisories from `/app/data/advisories.json`, resolves all dependencies using Minimal Version Selection per the policy in `/app/config/policy.json`, then writes a comprehensive audit report to `/app/output/dep_audit.json`.

The resolution algorithm, coupling metrics, vulnerability propagation rules, and the complete output schema are documented in `/app/docs/MVS_SPEC.md`, `/app/docs/METRICS_SPEC.md`, and `/app/docs/OUTPUT_FORMAT.md`. The project structure has stub files under `/app/cmd/depaudit/` and `/app/internal/` — build with `go build -o /app/depaudit ./cmd/depaudit/` and run the binary to produce the report. All floating-point values in the output are rounded to 6 decimal places, with sorted keys at every nesting level, 2-space indentation, and a trailing newline.
